claude-sfx gives Claude Code a voice. Every tool call, file edit, search, git push, and agent dispatch gets a distinct sound — synthesized from math, not audio files.

## Why audio feedback?

When an AI agent reads, writes, searches, and deploys on your behalf, you lose visibility. Audio feedback restores awareness:

- **Accessibility** — hear state changes without watching the terminal
- **Flow** — know a test passed without context-switching
- **Presence** — the agent feels like a collaborator, not a black box
